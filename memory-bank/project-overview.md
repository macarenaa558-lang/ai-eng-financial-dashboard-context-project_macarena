# Project Overview

## Purpose

This project is a financial dashboard that displays key financial indicators and charts based on income and outcome movements.

The application is composed of a React frontend and a FastAPI backend.

## Main capabilities

- Display total income and total outcome.
- Calculate profit and profit percentage.
- Group financial movements by month.
- Compare financial periods.
- Identify top financial categories.
- Detect unusual increases in outcomes.
- Filter financial data by date, category, operation type, and business type.

## Current data source

The backend currently generates deterministic mock financial movements.

There is no database or persistence layer.

## Main users

The dashboard is intended for users who need a visual overview of business financial performance.
